## ğŸ›¼ <b>How to run</b>  </div>

### [Back-End]

#### clone repository

```shell
$ git clone https://github.com/Jeeyeonn/innovation_clone_2_BE
```

#### into the repository

```shell
$ cd innovation_clone_2_BE
```

#### build

```shell
$ ./build gradle
```

#### run server

```shell
$ java -jar build/libs/InnovationCloneBeApplicationTests.jar
```

### [Front-end]

#### clone repository

```shell
$ git clone https://github.com/k-gardn/osulloc_FE
```

#### into the repository

```shell
$ cd osulloc_FE
```

#### install packages

```shell
$ yarn
```

#### start app

```shell
$ yarn start
```
